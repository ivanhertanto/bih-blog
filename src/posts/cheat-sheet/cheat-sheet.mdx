**SSH Key Gen**

```bash
ssh-keygen -t rsa -b 4096 -C "user id or email"
```

<br />

**Check Key**

```bash
ssh -T git@github.com
```

<br />

**MD5deep**

```bash
md5deep -rels * > checksums.md5
md5deep -rx checksums.md5 *
```

<br />

**Tmux**

```bash
    tmux setting
    tmux set -g mouse on \
    tmux set -g history-limit 9999
    tmux show -g | cat > ~/.tmux.conf
```
